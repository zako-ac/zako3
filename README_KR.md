## Zako는 무엇인가요?
**Zako**는 기본적으로 디스코드 음악 봇과 TTS 봇을 결합한 프로그램입니다. 디스코드 음성 채널에 다양한 오디오 트랙을 제공합니다.

## 제작 동기
디스코드에는 마이크를 사용하지 않고 목소리를 드러내지 않으려는 사람들이 많습니다. 이들은 보통 마이크를 끄고 채팅창을 통해서만 대화합니다. 하지만 대부분의 사람들은 다른 일을 하고 있을 때 채팅창을 항상 볼 수 있는 것은 아닙니다. 따라서 개발자들은 TTS(텍스트 음성 변환) 기술을 사용하여 채팅창을 읽어주는 봇을 만들었습니다. 이러한 봇들은 매우 잘 만들어졌지만, 저는 몇 가지 문제점과 개선 방안을 발견했습니다.
### 문제점
- **블로킹** 현재 메시지를 다 읽을 때까지 다음 메시지를 읽지 않습니다.
- **맞춤 설정 기능 부족** 사용자가 사용할 수 있는 TTS 음성이 제한적입니다. 사용자는 음성을 직접 설정할 수 없습니다.
